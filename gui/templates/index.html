MedLinker is a neuro-symbolic pipeline for biomedical entity linking — mapping mentions of diseases, chemicals, and other biomedical concepts in text to their standardized identifiers in the MeSH (Medical Subject Headings) ontology.

1 Linguistic Rules — entity extraction from raw text
2 Candidate Generation — string & semantic search over MeSH
3 Domain Rules — re-ranking with biomedical heuristics
4 LLM Disambiguation — final selection via language model

Built as a reproduction and extension of BioLinkerAI (PAKDD 2025) with added SapBERT embeddings, hybrid scoring, abbreviation expansion, and string normalization.
